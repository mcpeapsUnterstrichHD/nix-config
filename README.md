# my dotfiles

## installation

***only for macos***

---

### cloning

```bash
git clone https://github.com/mcpeapsUnterstrichHD/dotfiles.git
```

### go to dotfiles

```bash
cd dotfiles
```

### change executable flag

```bash
chmod +x install.sh && chmod +x update.sh
```

### install

```bash
./install.sh
```

## update dotfiles

### go to dotfiles

```bash
cd dotfiles
```

### update

```bash
./update.sh
```
